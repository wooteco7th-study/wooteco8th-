# java-convenience-store-precourse

## 📝️ 기능 목록

### ▪️ 입력 기능

- 파일 입력 받기
    - [x] 상품 목록 입력받기
    - [x] 행사 목록 입력받기

- [x] 구매할 상품명 및 수량 입력 받기
    - [x] 공백일 경우 예외를 발생시키기
    - [x] 지정된 입력 형식(`[콜라-10],[사이다-3]`)이 아닐 경우 예외를 발생시키기

- [x] 프로모션 행사 상품 추가 제공 여부 입력 받기
    - [x] 공백일 경우 예외를 발생시키기
    - [x] 지정된 입력 형식(`Y/N`)이 아닐 경우 예외를 발생시키기

- [x] 프로모션 할인 미적용 동의 여부 입력 받기
    - [x] 공백일 경우 예외를 발생시키기
    - [x] 지정된 입력 형식(`Y/N`)이 아닐 경우 예외를 발생시키기

- [] 멤버십 할인 여부 입력 받기
    - [] 공백일 경우 예외를 발생시키기
    - [] 지정된 입력 형식(`Y/N`)이 아닐 경우 예외를 발생시키기

- [] 상품 추가 구매 여부 입력 받기
    - [] 공백일 경우 예외를 발생시키기
    - [] 지정된 입력 형식(`Y/N`)이 아닐 경우 예외를 발생시키기

### ▪️ 보조 기능

- [] 잘못된 값이 들어올 시 예외 메세지 출력 후 재입력 받기
- [] 구분자를 기준으로 파싱하기

### ▪️ 비지니스 기능

- 재고 관리
    - [x] 재고 저장
    - [x] 구매 수량 재고에서 차감
        - 프로모션 할인 조건에 해당되는 경우, 프로모션 재고 차감
        - 프로모션 재고 부족한 경우 일반 재고 차감
    - 조건 미충족 시 예외를 반환한다.
        - [x] 구매 하려는 수량이 재고 수량을 초과하면 안된다.

- 프로모션 할인
    - [x] 프로모션 적용 여부 확인
        - 구매 날짜가 프로모션 기간 내에 포함되는지 확인
        - 구매 상품이 프로모션 적용 상품인지 확인
    - [x] 프로모션 적용 제품 추가 제공 여부 확인

- 멤버십 할인
    - [] 프로모션 미적용 금액의 30% 할인 적용
    - [] 최대 한도 `8,000원`

- 계산
    - [] 구매 상품마다 수량 및 금액 계산
    - [] 구매 상품 총 수량 및 총 금액 계산
    - [] 프로모션에 의해 할인된 금액 계산
    - [] 멤버십 할인 금액 계산
    - [] 최종 결제 금액 계산

### ▪️ 출력 기능

- [x] 환영 인사와 함께 상품명, 가격, 프로모션 이름, 재고 출력
    - [x] 재고 0개면 `재고 없음` 출력
- [] 구매 상품 내역, 증정 상품 내역, 금액 정보 출력
    - 영수증 포맷 사용
- [] 예외 메세지 출력
    - `[ERROR]` 로 시작할 것
